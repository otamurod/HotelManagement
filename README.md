# Hotel App

Multi-module Android app built on top of Clean Architecture using MVVM design pattern.

## Technology Stack

- Kotlin
- Kotlin Coroutines
- Flow and LiveData
- Dagger Hilt
- Retrofit
- Rest API
- MVVM
- Adapters
- Clean Architecture
- Multi-modularity (as recommended)
- And more...
